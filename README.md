# JEEDOM-Widget-Linky_V4
 Widget pour Jeedom en V4 permettant de regrouper les informations de consommation électrique Linky

<img src="/doc/linkyV4.gif" alt="linky"/>

Pour fonctionner les commandes doivent respecter le nommage suivant:
1) **Conso Annuelle**
2) **Conso Mensuelle**
3) **Conso Jour**
4) **Conso Horaire**
